# CryptoShield Intelligence Platform

**Professional-grade wallet risk analysis for Ethereum.**

CryptoShield helps crypto platforms meet AML/KYC compliance requirements
by automatically screening wallet addresses for sanctions exposure,
indirect contamination, and behavioral red flags.

---

## Features

- **Direct Sanctions Screening** — checks against OFAC SDN, confirmed exploiter addresses
- **Multi-Hop Graph Analysis** — traces fund flows up to 3 hops deep
- **Behavioral Pattern Detection** — detects structuring, layering, rapid bursts
- **Risk Scoring** — 0–100 score with CRITICAL / HIGH / MEDIUM / LOW verdict
- **JSON Export** — machine-readable reports for integration with compliance systems

---

## Setup

### 1. Clone the project

```bash
git clone <your-repo-url>
cd cryptoshield
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure your API key

```bash
cp .env.example .env
```

Open `.env` and set your Etherscan API key:

```
ETHERSCAN_API_KEY=your_key_here
```

Get a free API key at [etherscan.io](https://etherscan.io/apis).

> ⚠️ **Never commit your `.env` file to version control.**
